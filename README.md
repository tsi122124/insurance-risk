# Insurance Risk Analytics & Predictive Modeling

## Project Overview

This project analyzes historical automobile insurance data from AlphaCare Insurance Solutions (ACIS) to identify low-risk customer segments, evaluate risk drivers, and develop predictive models for risk-based pricing.

## Objectives

- Perform exploratory data analysis (EDA)
- Validate business hypotheses using statistical testing
- Build predictive models for claim severity and premium optimization
- Implement data version control using DVC
- Deliver actionable business recommendations
